# 功能

- 有数据库的轻量问答插件
- 支持模糊问答（未完成）
- 支持正则问答（未完成）
- 自动转译 CQ 码
- 支持图片回复
- 支持指令大杂烩
- 各个会话问答独立不影响
- 超管可添加全局词条，对所有会话生效
- 可以迁移问答到别的会话中（未完成）
- 支持概率触发问答 和 多答句概率回复（未完成）

# 安装

```
pip install nonebot-plugin-word-bank3
```

# 导入

```py
nonebot.load_plugin("nonebot_plugin_tortoise_orm")  # 务必最先导入数据库
nonebot.load_plugin("nonebot_plugin_word_bank3")
```
