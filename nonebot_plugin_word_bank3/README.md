# 操作指令
```
以下例句中未特别声明账号的则代表与用户qq为114514的对话
添加
    #[qg][qgzz][qgmh][zz][mh]问句 答句
    (qg代表全局，zz代表正则，mh代表模糊)
    例:#早上好 你好

    >>#早上好 你好
    问答已添加,编号为1。

    例:#qg中午好 你好

    >>#qg中午好 你好
    问答已添加,编号为2。

    例:#zz下午好 你好

    >>#zz下午好 你好
    问答已添加,编号为3。

    例:#mh晚上好 你好

    >>#mh晚上好 你好
    问答已添加,编号为4。

反馈
    >>早上好
    你好

    (此条为在群号为1314520的会话)
    >>中午好
    你好

    >>科宝下午好！
    你好

    >>晚上好，科宝！
    你好

删除
    #key[问句]/id[问句id] -r
    例:
        >>#key早上好 -r
        问答1已成功删除

    例:
        >>#id2 -r
        问答2已成功删除

查询
    ## 问句
    例：
        >>## 早上好
        问题 1."早上好"的回答如下:
        1. 你好 (回答id.答句)

    ## ~ 答句
    例:
        >>## ~ 你好
        问题"你好"的回答如下:
        1. 早上好 (问句id.问句)
        2. 中午好
        3. 下午好
        4. 晚上好

    查询教学行为(教学行为时间由env中的last_operation_time决定，单位为分钟，默认数值为10):
    ## -v
    例:
        >>## -v
        1.[添加- 1 秒前:]问题:早上好,回答
        [1]-1 . 你好 
        2.[添加- 1 秒前:]问题:中文好,回答
        [1]-1 . 你好 
        3.[添加- 1 秒前:]问题:下午好,回答
        [1]-1 . 你好 
        4.[添加- 1 秒前:]问题:晚上好,回答
        [1]-1 . 你好 
迁移
    （该操作没有检测是否在群中或拥有对方好友，请输入时再三检查是否打错，以及mg与mp迁移的对象是否正确）

    迁移至群:
    #key[问句]/id[问句id] -mg[群号]
    例:
        （将id1的问句迁移至群1314520）
        >>#id1 -mg1314520
        问题1已成功迁移至1314520

    迁移至人:
    #key[问句]/id[问句id] -mp[目标qq号]
    例:
        （将id1的问句迁移至用户114514）
        >>#id1 -mp114514
        问题1已成功迁移至114514
id范围操作
    适用于任何可以使用id进行操作的指令
    将对应指令的id[问句id]改为idx..idy即可
    例:
        批量删除
        >>#1..4 -r
        问答1已成功删除
        问答2已成功删除
        问答3已成功删除
        问答4已成功删除

        批量迁移
        >>#1..4 -mp114514
        问题1已成功迁移至114514
        问题2已成功迁移至114514
        问题3已成功迁移至114514
        问题4已成功迁移至114514
